const char * serviceUuid,
const char * characteristicUuid,
const char * dataToSend )

Fonction pour envoyer des données via une caractéristique BLE.

Cette fonction découvre le service BLE spécifié, recherche la caractéristique, puis tente d'envoyer les données fournies.

Paramètres
central- L'objet représentant l'appareil central BLE.
serviceUuid- UUID du service BLE à découvrir.
characteristicUuid- UUID de la caractéristique BLE à utiliser pour l'envoi des données.
dataToSend- Les données à envoyer sous forme de chaîne de caractères.
Renvoie
bool - Retourne true si les données ont été envoyées avec succès, sinon false.

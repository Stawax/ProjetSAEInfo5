Description détaillée

Header des icons à afficher sur l'écran LCD.

Images can be converted to XBM format by using the online converter here:

https://www.online-utility.org/image/convert/to/XBM

The output must be pasted in a header file, renamed and adjusted to appear as as a const unsigned char array in PROGMEM (FLASH program memory).
